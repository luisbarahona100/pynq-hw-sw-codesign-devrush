# Reconfiguring AMD FPGAs Using Python

## HW/SW Co-Design Workflow with PYNQ

This repository contains the material used during the Dev Rush inter-university event (Ecuador–Peru).

The objective of this session is to demonstrate how AMD FPGA-based heterogeneous SoCs can be reconfigured and controlled directly from Python using PYNQ.

Topics covered:

* FPGA acceleration using Vitis HLS
* HW/SW co-design workflow
* Vivado block design integration
* PYNQ overlays
* Runtime FPGA reconfiguration
* Jupyter notebook control
